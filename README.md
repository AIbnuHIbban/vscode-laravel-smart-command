 - Create Controller
 - Create Migration
 - Create Model + with option (also create migration and controller)
 - Create Seeder
 - Run Migrate
 - Run Migrate Rollback
 - Run Migrate Refresh
 - Start Server
 - Stop Server
 - Restart Server
